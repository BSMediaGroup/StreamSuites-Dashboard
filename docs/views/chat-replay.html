Historical Chat Replay (Planned)

This placeholder documents the intended layout for a future chat replay view. The final implementation will remain static and read-only, consuming exported chat logs without controlling live bots or altering runtime behavior.

Planned UI Blocks

  • Chat log source selector (file picker or URL input)
  • Stream lookup field (e.g., by stream_id)
  • Chronological message list with timestamp and author labels
  • Playback controls for pacing (play, pause, seek, speed)
  • Platform indicator badges for multi-platform logs

Notes

This view will not modify data, trigger bots, or manage authentication. It is intended for visualization of previously captured chat activity only.
