Historical Chat Replay

Planned Capability

This view will provide a read-only interface for replaying previously captured livestream chat logs without interacting with live bots or altering runtime behavior.

Planned UI Blocks

  • Chat log source selector (file picker or URL input)
  • Stream lookup field (e.g. by stream_id)
  • Chronological message list with timestamps and author labels
  • Playback controls (play, pause, seek, speed)
  • Platform indicator badges for multi-platform logs

Constraints

  • No data modification
  • No bot commands or outbound requests
  • No authentication or permissions logic
  • Visualization only
