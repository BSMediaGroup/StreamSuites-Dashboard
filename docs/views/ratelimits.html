Rate Limits

About Rate Limits

Rate limits define how frequently the runtime is allowed to perform actions such as sending messages, responding to chat triggers, polling APIs, or executing automated jobs.

This dashboard view is configuration-only. It does not enforce limits or interact with live bots. All values authored here are exported as JSON and consumed by the StreamSuites runtime.

Think of this page as a contract: it defines intent, not behavior.
