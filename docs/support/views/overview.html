Desktop Admin companion

StreamSuites ships a Desktop Admin tool with feature parity to the web dashboard plus local-only capabilities like multi-runtime management and inline stdout/stderr log viewing. The log viewer is read-only (not a terminal emulator), and operators can open an external terminal for interactive troubleshooting when needed. It runs locally alongside the runtime and syncs through the same exported state, so changes made on desktop appear on the web after the next refresh cycle.

Contact support for the latest Desktop Admin installer and onboarding steps.
